transform

The transform CSS property lets you rotate, scale, skew, or translate an element. It modifies the coordinate space of the CSS visual formatting model.

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

Value Description Demo
none Defines that there should be no transformation demo
matrix(n,n,n,n,n,n) Defines a 2D transformation, using a matrix of six values demo
matrix3d (n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) Defines a 3D transformation, using a 4x4 matrix of 16 values demo
translate(x,y) Defines a 2D translation demo
translate3d(x,y,z) Defines a 3D translation demo
translateX(x) Defines a translation, using only the value for the X-axis demo
translateY(y) Defines a translation, using only the value for the Y-axis demo
translateZ(z) Defines a 3D translation, using only the value for the Z-axis demo
scale(x,y) Defines a 2D scale transformation demo
scale3d(x,y,z) Defines a 3D scale transformation demo
scaleX(x) Defines a scale transformation by giving a value for the X-axis demo
scaleY(y) Defines a scale transformation by giving a value for the y-axis demo
scaleZ(z) (in com) Defines a 3D scale transformation by giving a value for the Z-axis demo
rotate(angle) Defines a 2D rotation, the angle is specified in the parameter demo
rotate3d(x,y,z,angle) Defines a 3D rotation demo
rotateX(angle) Defines a 3D rotation along the X-axis demo
rotateY(angle) Defines a 3D rotation along the Y-axis demo

skewX(angle) Defines a 2D skew transformation along the X-axis demo
skewY(angle) Defines a 2D skew transformation along the Y-axis demo
